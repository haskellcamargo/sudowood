# Sudowoodo
## Sudowoodo in your console!

## Usage

`$ woodo` - Outputs that it looks like a tree
`$ sudo woodo` - Outputs **sudowoodo**!

## Installation

Move the binary `woodo` to `/etc/bin`.

## Compiling (without optimization)

Run `ghc Woodo.hs`.
